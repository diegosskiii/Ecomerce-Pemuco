**E-Commer Pemuco**

Servicio de página web tipo Ecomerce, el cual va en ayuda para tener un seguimiento de ventas, conocer stock de productos y agilizar procesos.

----------------------------------------------------------------------------------------------------------------------------------------------

**MODULOS**

**Carrito de compra:**
* Agregar producto
* Eliminar producto
* Modificar producto
* Pagar

**Producto:**
* Agregar producto
* Eliminar producto
* Modificar producto
* Mostrar stock
* Ofertas 

**Venta:**
* Estado venta 
* Dashboard  
* Despacho o Retiro
* Generar boleta
	
	
**Usuario:**
* Registro
* Inicio de sesión
* Historial de compras
* Sistema de roles 
* Recuperar contraseñas
